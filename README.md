## 📌 진행 방식
- 스터디는 **매주 수요일 오후 10시**에 진행한다.
- 스터디 날에는 Issue 에 있는 부분을 보면서 각자의 의견을 공유한다. 혹은 추가로 공유하고 싶은 내용이 있다면 회의시간에 공유한다.
  - 의견에는 정답이 없기 때문에 자유롭게 이야기한다.
  - ISSUE comment 남길 시 주의사항
    - ❌ : 공감하지 않거나 다르게 생각하는 경우
    - ✅ : 공감하는 경우
    - 💬 : 또 다른 의견 제시나 생각을 작성하는 경우
